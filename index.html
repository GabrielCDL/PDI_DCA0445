Questão 1: Conversão para Escala de Cinza

O código abaixo converte uma imagem colorida para escala de cinza:


import cv2
img = cv2.imread('imagem.jpg')
cinza = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
cv2.imshow('Cinza', cinza)
cv2.waitKey(0)
cv2.destroyAllWindows()
